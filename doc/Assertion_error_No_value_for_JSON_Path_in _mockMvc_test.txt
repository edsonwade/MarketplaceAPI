Error : Assertion error: No value for JSON Path in JUnit test. when where list all elements of entity

Solution :  .andExpect(MockMvcResultMatchers.jsonPath("$[0].name").value("Bordeaux"))
            .andExpect(MockMvcResultMatchers.jsonPath("$[0].cost").value(10.55));
