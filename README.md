### Definition of TDD (Test driven development)

A software development process that encourages developers to write tests
for theis code before writing the code itself.

### TDD Process

     1 - Write Test
     2 - See Test Fail
     3 - Write Code
     4 - See Test Pass
     5 - Refactor
